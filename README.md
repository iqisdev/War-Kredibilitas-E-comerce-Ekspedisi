# War Kredibilitas E-commerce & Ekspedisi

## Deskripsi
Project analisis Social Network Analysis (SNA) untuk mengukur kredibilitas akun terkait topik e-commerce dan ekspedisi menggunakan Python dan Streamlit.

## Teknologi
- Python
- Streamlit
- Pandas
- NetworkX
- Plotly

## Cara Menjalankan

pip install -r requirements.txt

streamlit run app.py
